We started this project to make sure that HMC has up-to-date documentation for all of its products. The product documentation revolves around 3 main categories:

  • Developer Notes
  • IT Manager Notes
  • Business Rules

In addition there will be a how-to section organized as posts or articles to produce something close to a knowledge base base.

This asset lives as a collection of documentation repositories in GitHub under an organization called HMCDOCS. This site is the public face for this organization and it has four sections:

  • ABOUT which is this page
  • POSTS which is a collection of public posts and articles about documentation and other HMC topics that can be shared with the rest of the world
  • REPOS which lists all the repositories and a descrition for each. Access requires authentication.
  • INSIGHTS which is a charts to show some analytics about our doc repositories and how they are being used
