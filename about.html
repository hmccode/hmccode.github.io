About HMC code

This is a public site for HMc's source code repositories. We will use this site to provide an overview of the source code repositories, write technical posts and provide quick answers to how we produce products.

This asset lives as a collection of source repositories in GitHub under an organization called HMCCODE. This site is the public face for this organization and it has four sections:

  • HOME which lists all the repositories and a descrition for each. Access requires authentication.
  • ABOUT which is this page
  • POSTS which is a collection of public posts and articles about documentation and other HMC topics that can be shared with the rest of the world
  • INSIGHTS which is a charts to show some analytics about our doc repositories and how they are being used
